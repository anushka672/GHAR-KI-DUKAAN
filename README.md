# 🏠 Ghar Ki Dukaan
### *From Home. With Purpose.*

A women-empowerment marketplace where talented women creators can sell their handmade products and offer fabric redesign services.

---

## 🌸 About
Ghar Ki Dukaan connects skilled women artisans directly with customers who appreciate handmade, authentic Indian crafts. Every purchase empowers a woman working from home.

---

## ✨ Features
- 🛍️ **Marketplace** — Browse handmade products filtered by category and state
- ♻️ **Redesign Request** — Upload old clothes and get them redesigned by skilled creators
- 👩‍🎨 **Creator Dashboard** — Manage products and track orders
- 🔐 **Authentication** — Secure signup/login for shoppers and creators
- 📍 **State-wise Filter** — Shop handmade products from specific Indian states

---

## 🛠️ Tech Stack
| Frontend | Backend | Database |
|----------|---------|----------|
| React.js | Node.js | MongoDB |
| Tailwind CSS | Express.js | Mongoose |
| React Router | JWT Auth | MongoDB Atlas |

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- MongoDB installed locally

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/anushka672/GHAR-KI-DUKAAN.git
cd GHAR-KI-DUKAAN
```

**2. Setup Backend**
```bash
cd backend
npm install
```

Create `.env` file in backend folder:
```
PORT=5000
MONGO_URI=mongodb://localhost:27017/gharkidukaan
JWT_SECRET=gharkidukaan_super_secret_key_2024
```
```bash
npm run dev
```

**3. Setup Frontend**
```bash
cd frontend
npm install
npm start
```

**4.** Open `http://localhost:3000` 🎉
